# Event-specific food offers

Implement the feature requested in issue #25 (https://github.com/hanneseklund/MassanApp/issues/25).

Summary from the issue:
> for each event, create a special menu with a theme related to the event, put in on top in the food menu

Requirements:
- For each event, add a special themed menu (a short set of food items) whose theme relates to that event.
- Render this special menu at the top of the food menu, above the existing menu sections.
- Should work for all events currently in the seed/event data.

When complete, move this task file to `agent/tasks_completed/` using `git mv`.
