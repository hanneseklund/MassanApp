# Apply Stockholmsmässan graphical style

Implement issue #7: https://github.com/hanneseklund/MassanApp/issues/7

Use https://stockholmsmassan.se/ as a template and create a cohesive style for
the app — same fonts, colors, and general styling.

## Steps
- Inspect https://stockholmsmassan.se/ for typography (font families, weights,
  sizes), color palette (primary, secondary, accent, neutrals, backgrounds),
  spacing, button/link styles, headings, and overall layout conventions.
- Capture design tokens (CSS variables or Tailwind config) reflecting those
  fonts and colors.
- Apply the style system consistently across existing views (landing, event
  list, event detail, my pages, tickets, newsletter, auth, etc.).
- Ensure mobile-first responsiveness is preserved.
- Verify accessibility (contrast ratios, focus states) still holds.
