# Food ordering function

Implement food ordering feature as specified in issue #15
(https://github.com/hanneseklund/MassanApp/issues/15).

Summary of requirements (see issue for full detail):
- Function is available when an event is selected.
- User can choose between 10 typical fast food menus, each with description
  and image.
- Delivery options:
  - Pick up food at a couple of pickup locations within the venue, OR
  - Choose a 30-minute timeslot at a couple of restaurants where the food
    will be served.
- User pays for the food directly in the app.
